# Manual de Instalación - Sistema de Gestión de Residencias

## Requisitos del Sistema
- **Sistema Operativo**: Windows 10 o superior (64-bits recomendado).
- **Espacio en Disco**: Mínimo 500 MB libres.
- **Memoria RAM**: 4 GB o superior.

## Pasos de Instalación

1. **Descargar el Instalador**
   Localice el archivo `Residencias-Setup-1.0.0.exe` proporcionado.

2. **Ejecutar el Asistente**
   Haga doble clic en el archivo descargado. Windows puede pedir permisos de administrador; acéptelos para continuar.

3. **Ubicación de Instalación**
   Por defecto, la aplicación se instalará en `Archivos de Programa`. Puede cambiar esta ubicación si lo desea.

4. **Accesos Directos**
   El instalador creará automáticamente un acceso directo en el Escritorio y en el Menú Inicio.

5. **Finalizar**
   Una vez completada la barra de progreso, la aplicación se iniciará automáticamente.

## Configuración Inicial

Al abrir la aplicación por primera vez:
1. Se creará automáticamente la base de datos local.
2. Inicie sesión con las credenciales de administrador predeterminadas (si se le han proporcionado) o registre el primer usuario siguiendo las instrucciones del administrador de sistemas.

## Solución de Problemas

- **La aplicación no abre**: Verifique que no hay un antivirus bloqueando el proceso `residencias-app.exe`.
- **Error de base de datos**: Asegúrese de tener permisos de escritura en la carpeta de instalación o ejecute la aplicación como Administrador.
